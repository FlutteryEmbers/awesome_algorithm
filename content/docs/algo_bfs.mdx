queue = deque([root])
while queue:
    node = queue.popleft()

## !!steps BFS on General Graph

The third book.

```python ! bfs.py
queue = deque([root])
visit = set()
while queue:
    node = queue.popleft()
    if node in visit:
        continue

    visit.add(node)
    for neighbor in nei[node]:
        if node not in visit:
            queue.append(node)
```

## !!steps Dijstrak's Algorithm

The fourth book, explores the aftermath of the wars.

```python ! bfs.js
heap = [(0, root)]
visit = set()
while heap:
    dist, node = heapq.heappop(heap)
    if node in visit:
        continue

    visit.add(node)
    for next_dist, neighbor in nei[node]:
        if node not in visit:
            heapq.heappush(heap, (dist + next_dist, neighbor))
```
